Smart Home Energy Auditor
A simple web app to track your home energy use and estimate your electricity bill.

Main Features
Easy Interface: Modern dark-themed design.
Step-by-Step: Add your appliance details to see your daily power usage.  
Analysis: Calculates your predicted monthly bill and shows your energy consumption with charts.

How to Run
Install requirements: pip install streamlit pandas
Start the app: streamlit run energy.py
